Shopify Bugs Report

  • Cannot create sub-directory in asserts directory

    below are the shopify project structure


      my-shop
      ├── assets/  / store all website images, css, js, fonts /
      ├── config/
      ├── config.yml
      ├── layout/
      ├── locales/
      ├── sections/
      ├── snippets/
      ├── templates/
      └── theme.lock

    if you want to create sub-folders under asstes/ , like this:

      assets
      ├── gift-card.js
      ├── gift-card.scss.liquid
      ├── image
      │   ├── ajax-loader.gif
      │   └── ico-select.svg.liquid
      ├── lazysizes.js
      ├── match-media.min.js
      ├── password.js
      ├── theme.js
      ├── theme.scss.liquid
      └── vendor.js

    create a image folder and store all image under this , everything looks perfect , right ?
    but…
    when you try to upload them using theme kit , you will get HTTP 422 error
